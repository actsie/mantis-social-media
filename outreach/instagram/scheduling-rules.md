## Comment Sessions (10/day)
Each comment session is a bundled action — not just a comment in isolation.

**Per session:**
1. Find a same-day post in niche (prioritize posts from today)
2. Drop the comment (draft → approval → post)
3. Like that post
4. Like 1-2 older posts from the same account

= ~2-3 likes per session → ~20 likes tied to comments across the day

**Window:** 6:00am – 11:00pm PST

**Anti-pattern rules (must all be satisfied each day):**
- At least 2 comment sessions must be within 90min of each other (natural cluster)
- At least one gap between sessions must be 3hrs+ (long human break)
- No evenly-spaced intervals — gaps should range from ~45min to ~4hrs
- Aim for 1-2 natural "pairs" and 1-2 long breaks per day

---

## Standalone Like Batches (2-3x/day)
Separate from comment sessions — browsing and liking different accounts, no comment.

**Per batch:** 5-6 likes, accounts not commented on that day
**Spacing:** Every 4-6hrs (fewer batches since comment sessions cover ~20 likes)
**Within batch:** Likes spaced 8-18 minutes apart — never back to back

= ~15 likes standalone

---

## Daily Total
- ~20 likes from comment sessions
- ~15 likes from standalone batches
- **~35 likes/day total**
- **10 comments/day**

---

## Pattern Check (before scheduling each day)
- Read `engagement-log.json` for last 48hrs
- If gaps from prior 2 days look uniform, shift today's times ±3-8min
- Flag if same hour is used 3+ days in a row for first comment of day
