## Instagram browser automation — method confirmed (Feb 25)
- **`act` with natural language descriptions times out** — do NOT use for IG automation
- **Correct method: snapshot + ref-based clicks** — reliable and fast
  1. `browser open` → navigate to post URL
  2. `browser snapshot` → get element refs (e.g. Like button = ref like `e22`)
  3. `browser act` with `{"kind": "click", "ref": "eXX"}` → executes cleanly
- Tested successfully: liked @11nailangeles11 post (DVMlTkiE4yb), count went 3→4 ✅
- This is the method to use for all automated IG likes and interactions
